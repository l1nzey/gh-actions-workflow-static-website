# gh-actions-workflow-static-website
A simple GitHub Pages deployment workflow example

**Link to deployed site on GitHub Pages:** [https://l1nzey.github.io/gh-actions-workflow-static-website/](https://l1nzey.github.io/gh-actions-workflow-static-website/)

Solution to the **"GitHub Pages Deployment"** exercise in **"DevOps Projects"** from [roadmap.sh](https://roadmap.sh/projects/github-actions-deployment-workflow).

This project demonstrates:

- Setting up a GitHub repository with a static website
- Using GitHub Actions to automatically deploy changes to GitHub Pages
- Continuous Deployment (CD) workflow for static content
